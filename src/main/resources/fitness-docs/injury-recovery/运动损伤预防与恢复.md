# 运动损伤预防与恢复

## 肩部损伤

肩关节是人体活动范围最大的关节，也是最容易受伤的关节之一。

### 常见肩部损伤
1. **肩袖损伤**：过度训练或动作不当导致肩袖肌群（冈上肌、冈下肌、小圆肌、肩胛下肌）受伤
2. **肩峰撞击综合征**：肩关节活动时肩峰与肱骨头之间的软组织被反复挤压
3. **肩关节不稳定**：韧带松弛或损伤导致肩关节稳定性下降

### 预防措施
- 充分热身，尤其是肩部环绕和肩袖激活
- 训练中保持肩胛骨稳定，避免耸肩代偿
- 加强肩袖小肌群训练（外旋、内旋练习）
- 避免过度使用大重量的过头推举
- 注意推拉训练比例，避免推多拉少的不平衡

### 康复训练
- 急性期：休息、冰敷、避免疼痛动作
- 恢复期：弹力带肩袖外旋，面拉，肩胛骨稳定训练
- 回归训练：逐步增加负荷，从小重量高次数开始

## 腰部损伤

腰椎是力量训练中负荷最大的部位之一，不良姿势容易导致腰部损伤。

### 常见腰部损伤
1. **腰椎间盘突出**：椎间盘纤维环破裂，髓核突出压迫神经
2. **腰肌劳损**：腰部肌肉长期过度使用导致慢性损伤
3. **骶髂关节紊乱**：骨盆区域关节功能异常

### 预防措施
- 所有训练动作中保持腰椎中立位
- 加强核心肌群训练（平板支撑、死虫式、抗旋转训练）
- 硬拉和深蹲时注意正确呼吸和腹内压
- 避免圆背进行任何负重动作
- 久坐人群注意髋屈肌拉伸

### 康复训练
- 急性期：避免弯腰负重，适当卧床休息
- 恢复期：麦肯基伸展、猫牛式、鸟狗式等核心稳定训练
- 回归训练：从体重动作开始，逐步过渡到负重训练

## 膝关节损伤

膝关节是训练中最常受压的关节，尤其在深蹲和跑步中。

### 常见膝关节损伤
1. **髌骨软化症**：髌骨软骨磨损退变，表现为膝前痛
2. **半月板损伤**：膝关节旋转或撞击导致半月板撕裂
3. **前交叉韧带损伤**：急停变向或着地不稳导致 ACL 损伤
4. **髂胫束摩擦综合征**：膝盖外侧疼痛，常见于跑步者

### 预防措施
- 深蹲时膝盖追脚尖方向，避免内扣
- 加强臀中肌训练，改善膝关节对线
- 注意股四头肌和腘绳肌的力量平衡
- 跑步时选择合适的鞋子和路面
- 训练前做好膝关节热身和激活

### 康复训练
- 急性期：RICE原则（休息、冰敷、加压、抬高）
- 恢复期：直腿抬高、靠墙静蹲、平衡板训练
- 回归训练：从低冲击运动开始（游泳、骑车），逐步恢复深蹲

## 手腕和手肘损伤

力量训练中手腕和手肘常因握姿不当或过度训练受伤。

### 常见损伤
1. **网球肘（肱骨外上髁炎）**：前臂伸肌群过度使用
2. **高尔夫球肘（肱骨内上髁炎）**：前臂屈肌群过度使用
3. **腕管综合征**：手腕反复屈伸或持续压力导致正中神经受压

### 预防措施
- 握杠时保持手腕中立，避免过度屈伸
- 使用护腕辅助大重量训练
- 训练前活动手腕关节，训练后拉伸前臂肌群
- 避免握力训练过度

### 康复训练
- 急性期：休息，冰敷，避免诱发动作
- 恢复期：前臂旋前旋后练习、离心腕屈伸、握力球训练
- 回归训练：逐步恢复握力需求高的训练

## 运动损伤通用处理原则

### 急性损伤处理（POLICE原则）
1. **P**rotect（保护）：避免加重损伤的动作
2. **O**ptimal **L**oading（适度负荷）：在不加重症状前提下进行适度活动
3. **I**ce（冰敷）：前48小时内每次15-20分钟
4. **C**ompression（加压）：弹性绷带适当加压
5. **E**levation（抬高）：抬高受伤部位减少肿胀

### 何时就医
- 疼痛剧烈无法活动
- 关节明显肿胀或畸形
- 持续疼痛超过两周未改善
- 活动时出现弹响伴随疼痛
- 肢体出现麻木或无力感
