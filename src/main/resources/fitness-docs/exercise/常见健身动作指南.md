# 常见健身动作指南

## 深蹲（Squat）

深蹲是最基础也是最重要的复合动作之一，主要锻炼股四头肌、臀大肌和核心肌群。

### 标准动作要点
1. 双脚与肩同宽或略宽，脚尖微微外展约 15-30 度
2. 挺胸收腹，脊柱保持中立位，目视前方
3. 屈髋屈膝同时进行，臀部向后坐，膝盖沿脚尖方向打开
4. 下蹲至大腿至少与地面平行（全蹲可更深），保持脚跟不离地
5. 起身时通过脚跟发力，臀部和大腿同时发力站起

### 常见错误
- 膝盖内扣：加强臀中肌训练，注意膝盖追脚尖
- 弯腰驼背：加强核心和上背力量，适当降低重量
- 脚跟抬起：检查踝关节灵活性，可垫小片杠铃片辅助

### 变体动作
- 颈前深蹲：杠铃放在锁骨前，更强调股四头肌
- 高脚杯深蹲：双手持壶铃或哑铃贴胸前，适合初学者
- 保加利亚分腿蹲：单腿训练，提升平衡和单侧力量

## 硬拉（Deadlift）

硬拉是全身性复合动作，主要锻炼后链肌群（竖脊肌、臀大肌、腘绳肌）。

### 标准动作要点
1. 双脚与髋同宽，杠铃位于脚中部正上方
2. 屈髋屈膝抓握杠铃，双手略宽于膝盖
3. 收紧背部，胸椎挺起，腰椎保持自然弯曲
4. 发力时腿部和背部同时发力，杠铃贴身上移
5. 锁定时髋部完全伸展，肩胛骨后收

### 常见错误
- 圆背拉起：重量过大或核心薄弱，需降低重量强化核心
- 杠铃远离身体：会增加腰椎压力，杠铃应始终贴身
- 锁定时过度后仰：容易伤腰，到位即可

### 变体动作
- 罗马尼亚硬拉：膝盖微屈，强调腘绳肌和臀部
- 相扑硬拉：宽站距，更多臀部和内收肌参与
- 六角杠硬拉：使用六角杠，降低对腰椎的压力

## 卧推（Bench Press）

卧推是训练上半身推力的核心动作，主要锻炼胸大肌、三角肌前束和肱三头肌。

### 标准动作要点
1. 仰卧在卧推架上，眼睛位于杠铃正下方
2. 五点支撑：双脚踩实地面，臀部、上背和头部紧贴凳面
3. 收紧肩胛骨，胸椎微微挺起形成自然拱背
4. 双手握距略宽于肩，解扣后将杠铃移至胸部正上方
5. 杠铃下放至胸部中下部（约乳头位置），轻触后推起

### 常见错误
- 肩膀前伸：肩胛骨未收紧，容易造成肩部损伤
- 杠铃弹胸：借力反弹，减少肌肉刺激且有受伤风险
- 臀部离凳：会增加腰椎压力，保持五点支撑

### 变体动作
- 上斜卧推：凳面倾斜 30-45 度，强调胸大肌上束
- 哑铃卧推：增加运动范围和稳定性训练
- 窄距卧推：强调肱三头肌

## 引体向上（Pull-up）

引体向上是经典的自重拉力动作，主要锻炼背阔肌、肱二头肌和核心稳定性。

### 标准动作要点
1. 双手正握（掌心向前）握杠，握距略宽于肩
2. 悬挂时充分伸展手臂，肩胛骨上旋
3. 发力时先沉肩（肩胛骨下沉），再弯曲手臂拉起
4. 拉至下巴超过杠面即可，上背充分收缩
5. 控制速度缓慢下放，充分拉伸背阔肌

### 常见错误
- 身体大幅摆动：借助惯性拉起，减少肌肉刺激
- 只弯曲手臂：没有启动肩胛骨运动，背部参与不足
- 下放过快：浪费离心阶段的肌肉刺激

### 变体动作
- 反握引体向上：掌心向自己，更多肱二头肌参与
- 澳式引体向上：身体倾斜，适合初学者入门
- 负重引体向上：腰间挂重量片，进阶训练

## 肩推（Overhead Press）

肩推是训练肩部力量的核心动作，主要锻炼三角肌和肱三头肌。

### 标准动作要点
1. 双脚与肩同宽，核心收紧保持身体稳定
2. 杠铃或哑铃位于锁骨前方，肘部略微向前
3. 推举时将重量沿直线向上推至头顶正上方
4. 杠铃过头后头部微微前移，使杠铃位于头顶正上方
5. 控制速度下放至起始位置

### 常见错误
- 过度挺腰：核心力量不足或重量过大，容易伤腰
- 推举轨迹偏前：杠铃应在锁定时位于头顶正上方
- 耸肩代偿：上斜方肌过度参与，注意沉肩发力

### 变体动作
- 坐姿肩推：减少腿部和核心借力
- 阿诺德推举：旋转手腕，增加三角肌刺激
- 单臂哑铃推举：提升单侧力量和核心稳定性
