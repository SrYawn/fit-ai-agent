# 训练计划制定指南

## 新手入门训练计划

适合零基础或训练经验不足3个月的初学者，以学习动作模式和建立训练习惯为主。

### 训练频率
- 每周 3 次全身训练，隔天进行
- 每次训练时间 45-60 分钟
- 每个动作 3 组，每组 10-12 次

### 推荐训练安排
**A日**：深蹲、卧推、杠铃划船、肩推、平板支撑
**B日**：硬拉、上斜卧推、引体向上（或高位下拉）、面拉、农夫行走

按 A-B-A / B-A-B 交替进行，每周交替。

### 新手注意事项
- 优先掌握正确动作模式，不急于加重
- 每次训练逐步增加重量（每次增加 2.5-5kg）
- 重视热身和拉伸
- 保证充足睡眠（7-9小时）和营养摄入

## 中级训练计划

适合有6个月以上训练经验，已掌握基本动作的训练者。

### 上下肢分化训练（4天/周）
将训练按上半身和下半身分化，每个部位每周训练2次。

**上肢A（推力为主）**：
1. 杠铃卧推 4×6-8
2. 上斜哑铃卧推 3×8-10
3. 坐姿肩推 3×8-10
4. 绳索飞鸟 3×12-15
5. 三头下压 3×10-12
6. 侧平举 3×12-15

**下肢A（股四为主）**：
1. 杠铃深蹲 4×6-8
2. 腿举 3×10-12
3. 保加利亚分腿蹲 3×10-12
4. 腿屈伸 3×12-15
5. 小腿提踵 4×15-20
6. 腹肌训练

**上肢B（拉力为主）**：
1. 引体向上 4×6-8
2. 杠铃划船 3×8-10
3. 坐姿绳索划船 3×10-12
4. 面拉 3×12-15
5. 锤式弯举 3×10-12
6. 反向飞鸟 3×12-15

**下肢B（后链为主）**：
1. 罗马尼亚硬拉 4×6-8
2. 臀推 3×10-12
3. 腿弯举 3×10-12
4. 髋外展 3×12-15
5. 小腿提踵 4×15-20
6. 腹肌训练

## 高级训练计划

适合有2年以上系统训练经验，力量水平达到中级以上的训练者。

### PPL分化训练（推/拉/腿，6天/周）

**推日（Push）**：
1. 杠铃卧推 5×5（主项，渐进超负荷）
2. 上斜哑铃卧推 4×8-10
3. 坐姿肩推 4×8-10
4. 绳索飞鸟 3×12-15
5. 侧平举 4×15-20（超级组）
6. 绳索下压 + 仰卧臂屈伸 3×12-15（超级组）

**拉日（Pull）**：
1. 硬拉 5×5（主项）
2. 加重引体向上 4×6-8
3. T杠划船 4×8-10
4. 面拉 3×15-20
5. 杠铃弯举 3×8-10
6. 锤式弯举 3×10-12

**腿日（Legs）**：
1. 杠铃深蹲 5×5（主项）
2. 前蹲/腿举 4×8-10
3. 罗马尼亚硬拉 3×10-12
4. 保加利亚分腿蹲 3×10-12
5. 腿屈伸 + 腿弯举 3×12-15（超级组）
6. 小腿提踵 4×15-20

## 减脂期训练调整

减脂期训练策略的核心是保持力量训练强度，维持肌肉量。

### 训练原则
- **保持训练强度**：不要大幅降低训练重量，尽量维持力量水平
- **适当减少训练容量**：可以减少组数（从4组降到3组），但保持重量
- **增加有氧训练**：每周2-4次有氧，每次20-40分钟
- **优先选择低冲击有氧**：快走、椭圆机、游泳、骑车

### 减脂期有氧训练安排
- **LISS（低强度持续有氧）**：心率维持在最大心率60-70%，持续30-45分钟
- **HIIT（高强度间歇训练）**：20秒全力 + 40秒休息，共15-20分钟
- HIIT 效率更高但恢复需求大，不宜每天进行
- 建议每周2次 HIIT + 2次 LISS

## 训练周期化

### 线性周期化（适合新手和中级）
- 每周训练重量线性增加
- 例如：深蹲每周增加 2.5kg
- 当进步停滞时进行 Deload（减载周）

### 波动周期化（适合中高级）
- 每周交替使用不同的强度和次数
- 例如：周一重量日 5×5，周四容量日 4×10
- 避免单一刺激导致适应停滞

### Deload（减载周）
- 每训练 4-6 周安排一个减载周
- 减载方式：重量降低 40-50%，保持组数和次数
- 或者：保持重量，组数减半
- 目的：消除累积疲劳，让身体恢复和适应
