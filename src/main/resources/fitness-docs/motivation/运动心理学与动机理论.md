# 运动心理学与动机理论

## 自我决定理论（Self-Determination Theory）

自我决定理论是理解运动动机的核心框架，由 Deci 和 Ryan 提出。该理论认为人类有三种基本心理需求：

### 自主性（Autonomy）
- 用户需要感到自己在训练中有选择权和控制感
- 激励策略：提供多种训练方案供选择，而非强制指定
- 避免命令式语气，使用"你可以尝试..."而非"你必须..."
- 当用户想调整计划时，支持而非否定

### 胜任感（Competence）
- 用户需要感到自己有能力完成训练目标
- 激励策略：设置渐进式目标，确保用户能体验到成功
- 关注进步而非绝对水平，强调"比上次进步了"
- 当用户遇到困难时，将挑战分解为更小的可达成步骤

### 归属感（Relatedness）
- 用户需要感到被理解、被支持
- 激励策略：表达对用户训练旅程的关注和理解
- 使用共情性语言，如"我理解这很不容易"
- 营造陪伴感，让用户感到不是一个人在坚持

## 变化阶段模型（Transtheoretical Model）

该模型描述了行为改变的五个阶段，每个阶段需要不同的激励策略：

### 前意向阶段（Precontemplation）
- 特征：用户尚未考虑开始运动
- 策略：提供运动益处的信息，不施加压力
- 话术示例："运动不仅能改善体型，还能提升睡眠质量和情绪状态"

### 意向阶段（Contemplation）
- 特征：用户在考虑开始运动，但还没行动
- 策略：帮助用户权衡利弊，降低开始的门槛
- 话术示例："哪怕每天只走15分钟，也是一个很好的开始"

### 准备阶段（Preparation）
- 特征：用户准备在近期开始运动
- 策略：帮助制定具体、可执行的计划
- 话术示例："我们可以从每周3次、每次30分钟的轻度训练开始"

### 行动阶段（Action）
- 特征：用户已经开始规律运动（6个月以内）
- 策略：强化正向行为，帮助应对障碍
- 话术示例："你已经坚持了两周，这个习惯正在形成中"

### 维持阶段（Maintenance）
- 特征：用户已经规律运动超过6个月
- 策略：预防倦怠，引入变化和新挑战
- 话术示例："你的坚持令人敬佩，也许可以尝试一些新的训练方式来保持新鲜感"

## 目标设定理论（Goal-Setting Theory）

### SMART 目标原则
- Specific（具体）：明确训练内容和标准
- Measurable（可衡量）：用数据追踪进度
- Achievable（可达成）：基于当前水平设定合理目标
- Relevant（相关）：与用户的核心动机关联
- Time-bound（有时限）：设定明确的时间节点

### 目标层次设计
- 长期目标（3-6个月）：提供方向感
- 中期目标（4-8周）：阶段性里程碑
- 短期目标（每周/每次训练）：即时可达成的小目标
- 过程目标 vs 结果目标：关注"完成训练"而非"减重X公斤"

## 内在动机与外在动机

### 内在动机培养
- 帮助用户发现运动本身的乐趣
- 关注运动带来的即时感受（精力充沛、心情愉悦）
- 鼓励探索不同运动形式，找到真正喜欢的

### 外在动机的合理利用
- 短期内外在激励（打卡、奖励）可以帮助建立习惯
- 长期应逐步转向内在动机驱动
- 避免过度依赖外在奖励导致动机内化失败
