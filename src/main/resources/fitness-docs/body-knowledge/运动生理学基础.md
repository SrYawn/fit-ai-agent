# 运动生理学基础

## 肌肉系统概述

人体有超过600块骨骼肌，了解主要肌肉群的功能有助于设计更有针对性的训练计划。

### 上半身主要肌群
- **胸大肌**：位于胸前，负责手臂的水平内收、屈曲和内旋。分为锁骨部（上胸）和胸肋部（中下胸）
- **背阔肌**：人体最大的背部肌肉，负责手臂的内收、伸展和内旋。V字体型的关键肌肉
- **三角肌**：覆盖肩关节的三角形肌肉，分为前束、中束和后束，负责手臂各方向的抬举
- **斜方肌**：从枕骨延伸到胸椎，分上中下三部分，负责肩胛骨的上提、后缩和下压
- **肱二头肌**：上臂前侧，负责肘关节屈曲和前臂旋后
- **肱三头肌**：上臂后侧，负责肘关节伸展，占上臂体积的2/3

### 下半身主要肌群
- **股四头肌**：大腿前侧最大肌群，负责膝关节伸展。包括股直肌、股外侧肌、股内侧肌、股中间肌
- **腘绳肌**：大腿后侧，负责膝关节屈曲和髋关节伸展。包括股二头肌、半腱肌、半膜肌
- **臀大肌**：人体最大最有力的肌肉，负责髋关节伸展、外旋和外展
- **臀中肌**：位于臀部外侧，负责髋关节外展和稳定骨盆
- **小腿三头肌**：包括腓肠肌和比目鱼肌，负责踝关节跖屈

### 核心肌群
- **腹直肌**：六块腹肌所在，负责脊柱屈曲
- **腹外斜肌和腹内斜肌**：负责躯干旋转和侧屈
- **腹横肌**：最深层腹肌，像腰带一样稳定躯干
- **竖脊肌**：脊柱两侧的深层肌群，负责脊柱伸展和稳定

## 肌肉生长原理

### 肌肉肥大的三大机制
1. **机械张力**：肌肉在负荷下产生的力学刺激，是肌肉生长最重要的驱动力
2. **代谢压力**：训练产生的代谢副产物（如乳酸、氢离子）积累，引发合成代谢信号
3. **肌肉损伤**：训练造成的微小肌纤维损伤，触发修复和生长过程

### 肌肉生长的条件
- **渐进超负荷**：逐步增加训练刺激（重量、次数、组数）
- **充足蛋白质**：提供肌肉合成所需的氨基酸原料
- **恢复休息**：肌肉在休息时修复和生长，训练只是提供刺激
- **激素环境**：睾酮、生长激素、胰岛素样生长因子等促进肌肉合成

### 肌纤维类型
- **I型（慢肌纤维）**：抗疲劳能力强，适合耐力活动，力量较小
- **IIa型（快肌纤维）**：兼具力量和耐力特性
- **IIx型（快肌纤维）**：爆发力最强，但容易疲劳
- 力量训练主要刺激 II 型肌纤维的生长

## 能量系统

人体有三大能量系统，供能方式决定了不同运动类型的能量来源。

### 磷酸肌酸系统（ATP-CP系统）
- 供能时间：0-10秒
- 特点：供能速度最快，但持续时间最短
- 适用运动：举重、短跑、跳跃等极短时间爆发性活动
- 恢复：完全恢复需要 3-5 分钟

### 糖酵解系统（无氧系统）
- 供能时间：10秒-2分钟
- 特点：供能速度较快，产生乳酸
- 适用运动：400米跑、高强度间歇训练、力量训练中高次数组
- 乳酸积累导致肌肉灼烧感和疲劳

### 有氧氧化系统
- 供能时间：2分钟以上
- 特点：供能持久但速度较慢，需要氧气参与
- 适用运动：长跑、游泳、骑行等耐力运动
- 主要燃料：脂肪和碳水化合物

## 恢复与适应

### 超量恢复原理
- 训练打破身体的稳态平衡
- 休息期间身体不仅恢复到原始水平，还会超越原始水平
- 在超量恢复的高点进行下一次训练，才能实现持续进步
- 训练过于频繁：恢复不足，可能导致过度训练
- 训练间隔过长：超量恢复消退，回到基线水平

### 影响恢复的因素
1. **睡眠**：最重要的恢复手段，建议7-9小时
   - 深度睡眠期间生长激素分泌最旺盛
   - 睡眠不足会增加皮质醇（分解激素），降低睾酮
2. **营养**：训练后及时补充蛋白质和碳水化合物
3. **压力管理**：慢性压力影响激素环境和恢复质量
4. **主动恢复**：低强度有氧、拉伸、泡沫轴滚动

### 过度训练的信号
- 训练表现持续下降
- 持续的疲劳和肌肉酸痛
- 情绪低落、失眠
- 容易生病（免疫力下降）
- 心率异常（静息心率升高）

## 柔韧性与关节活动度

### 柔韧性的重要性
- 保持良好的关节活动范围，确保动作质量
- 降低运动损伤风险
- 改善姿态和日常生活功能
- 不是越软越好，稳定性同样重要

### 提升柔韧性的方法
1. **静态拉伸**：在运动后进行，每个动作保持 20-30 秒
2. **动态拉伸**：在运动前进行，通过运动范围内的动态动作热身
3. **PNF拉伸**：收缩-放松技术，效果最好但需要经验
4. **泡沫轴滚动**：自我筋膜放松，缓解肌肉紧张

### 常见活动度受限及改善
- **踝关节背屈受限**：影响深蹲深度，可通过踝关节动态拉伸和小腿拉伸改善
- **髋关节屈曲受限**：影响深蹲和硬拉，可通过90/90拉伸和蝴蝶式改善
- **胸椎伸展受限**：影响肩推和卧推，可通过泡沫轴胸椎伸展改善
- **肩关节外旋受限**：影响深蹲杠铃位置和卧推稳定性，可通过肩袖拉伸改善
